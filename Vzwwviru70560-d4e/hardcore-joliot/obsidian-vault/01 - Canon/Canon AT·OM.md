# Canon AT·OM

> Regles fondamentales et immuables du systeme AT·OM

---

## Principes Fondamentaux

### 1. Souverainete Humaine (Rule #1)
L'humain reste toujours aux commandes. Aucune decision automatique ne peut etre prise sans validation humaine explicite.

### 2. Zero Communication IA-vers-IA (Rule #4)
Les agents ne communiquent JAMAIS directement entre eux. Toute interaction passe par l'humain ou le systeme central.

### 3. Pas de Classement (Rule #5)
Aucun agent ne peut etre classe, note ou compare a un autre. Chaque agent a une fonction unique et egale.

### 4. Tracabilite Totale (Rule #6)
Chaque action d'agent est tracee et peut etre auditee. L'humain peut toujours comprendre pourquoi une decision a ete prise.

## Le Sceau (Activation)

Le Sceau est le rituel d'engagement souverain:

1. **Duree**: Exactement 2 secondes de contact tactile
2. **Animation**: Frequence monte de 432Hz a 999Hz
3. **Confirmation**: Explosion de 350 particules (L'Essaim)
4. **Session**: Creation d'une session authentifiee

## Brise-Circuit (Kill-Switch)

Le Brise-Circuit permet la dispersion totale en cas d'urgence:

### Niveaux de Dispersion
1. **Partiel (20%)** - Desactive les agents non-essentiels
2. **Sectoriel (50%)** - Desactive une sphere complete
3. **Total (100%)** - Dispersion de tous les agents

### Activation
- Route: `/protocol-999`
- Code de confirmation requis
- Autorisation humaine obligatoire
