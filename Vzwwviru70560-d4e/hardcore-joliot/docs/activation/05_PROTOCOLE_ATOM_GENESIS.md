### Détail de Chaque Phase

#### Phase 1 : Installation Physique

```
Objectif : Être stable, sécurisé, prêt

Actions :
□ Choisir position stable (pas de sable direct)
□ Vérifier visibilité écran (ombre si soleil levant)
□ Placer protection matériel
□ Tester ouverture laptop et connexion batterie
□ Confirmer application CHE-NU accessible
```

#### Phase 2 : Silence (5 minutes)

```
Objectif : Transition état actif → état réceptif

Actions :
□ Fermer les yeux 30 secondes
□ 5 respirations profondes
□ Observer l'environnement sans jugement
□ NE PAS toucher le clavier
□ NE PAS interpréter (pensées, signes, etc.)
```

#### Phase 3 : Activation Technique

```
Objectif : Initialiser le système CHE-NU

Actions :
□ Ouvrir navigateur / application
□ Se connecter : admin@chenu.com / diamant999
□ Accéder à /admin/cortex
□ Vérifier Neural Swarm visible (350 points)
□ Ouvrir Command Center
□ Taper et envoyer le PREMIER ORDRE GLOBAL
```

#### Phase 4 : Clôture

```
Objectif : Terminer proprement sans fanfare

Actions :
□ Fermer l'application
□ Éteindre ou mettre en veille
□ Ranger dans housse de protection
□ Quitter le lieu calmement
□ NE PAS poster sur réseaux sociaux
□ NE PAS appeler quelqu'un immédiatement
```

## 5.5 Variantes et Cas Particuliers

### Si le réseau fonctionne

Procéder normalement. La synchronisation cloud se fera automatiquement.

### Si le réseau est absent

```
Mode Souverain activé automatiquement.
L'ordre est stocké localement.
Synchronisation au retour de connexion.
Le Cycle 0 reste valide (timestamp local fait foi).
```

### Si une erreur technique survient

```
1. Ne pas paniquer
2. Prendre une capture d'écran de l'erreur
3. Noter l'heure exacte
4. Tenter UN redémarrage de l'application
5. Si échec : clôturer et documenter
6. Le Cycle 0 peut être reporté sans conséquence
```

### Si conditions météo défavorables

```
Report autorisé. Aucune pénalité.
Prochaine fenêtre : aube suivante ou retour conditions OK.
L'important est la sécurité matérielle et personnelle.
```
